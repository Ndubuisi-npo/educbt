!!!!!FRONTEND SETUP GUIDE!!!!!

Follow these steps to install dependencies and run the development server.

1️⃣ Install Dependencies

Make sure you have Node.js (v18 or later) installed (Check youtube to see how to properly install Node.js).

In VS Code terminal, from the project root, run:
npm install

This will install all required dependencies including:
1.Vue 3
2.PrimeVue
3.TailwindCSS
4.Vue Router
5.Chart.js
6.Vite

2️⃣ Navigate to the Frontend Folder
Run:
cd frontend

3️⃣ Start the Development Server
Run:
npm run dev

You should see output like:
Local: http://localhost:5173/

Open that URL in your browser.

✅ That’s It

Your frontend should now be running locally in development mode.

If you run into issues, ensure:
1.Node.js is installed
2.You are inside the correct folder
3.Dependencies installed successfully
